Scan & Go Local+ — DEMO PACK

Fichiers inclus :
- sgl_index.html — page d’accueil
- sgl_studio.html — studio pour créer une page (génère URL + QR + CSV)
- sgl_profile.html — page publique (rendue depuis les paramètres de l’URL)
- sgl_cards.html — générateur de cartes QR (A6) imprimables
- sgl_clients_template.csv — modèle Google Sheets pour vos clients

Mode d’emploi en 3 min :
1) Ouvrez sgl_studio.html et remplissez les champs (nom, texte, lien de don).
2) Cliquez “Générer le lien” → prévisualisez la page → imprimez le QR (ou copiez l’URL).
3) Ouvrez sgl_cards.html pour générer des cartes A6 à poser partout.

Dons : utilisez Stripe Payment Links / PayPal.Me / Lydia. Le paiement est traité par le prestataire choisi.
Nous ne collectons aucune donnée bancaire.

Astuce : pour montants rapides (2/5/10€), laissez un lien générique (montant libre). La page ajoutera automatiquement ?amount=2/5/10 s’il n’existe pas déjà.
